# Number Guessing Game (Python)

A simple and interactive number guessing game written in Python.  
The player must guess a randomly generated number between **1 and 100**, while the score decreases with each incorrect attempt.

---

## üéÆ How to Play
1. Run the program.
2. Enter a number between **1 and 100**.
3. The game will tell you if your guess is **too high** or **too low**.
4. Each wrong guess reduces your score by **10 points**.
5. Press **q** at any time to quit the game.
6. After winning, you can choose to play again.
